# SOMA Recommender

SOMA Recommender is a project for recommending SW Maestro lectures and learning content.

## Project Structure

- `soma-recommender-backend/`: Python backend and recommendation agent workspace
- `soma-recommender-extension/`: Browser extension client

## Tech Stack

- Backend: Python 3.11, uv, FastAPI, pytest, ruff, pyright
- Extension: JavaScript, Chrome Extension, ESLint, Prettier
